## libWinForm
### Installation  
Create a folder in your PowerShell modules folder and place the module file within. The folder and module file must have the same name.
### Sample Usage  
Import the module.  
```
Import-Module wtsapi
```
MessageBox
```
System.Windows.Forms.MessageBox -Text 'Hello'
```
